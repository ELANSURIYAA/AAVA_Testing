**Expected Outcome:** Returns top 5 customers per region with correct aggregations, sales, and product info.

## Test Case 2
**Test Case ID:** TC02  
**Description:** Edge case - Customer with no sales in 2023.  
**Expected Outcome:** Customer appears with zero sales, zero orders, empty product list.

## Test Case 3
**Test Case ID:** TC03  
**Description:** Edge case - Sale with NULL discount_applied field.  
**Expected Outcome:** Sale is excluded from aggregation and performance CTE.

## Test Case 4
**Test Case ID:** TC04  
**Description:** Edge case - Sale with sale_metadata missing 'country' or 'source' field.  
**Expected Outcome:** Sale is excluded from sales_performance if 'country' is missing or not 'USA'; 'source' is NULL in output.

## Test Case 5
**Test Case ID:** TC05  
**Description:** Edge case - Sale with boundary sale_amount values (exactly 1000, 500).  
**Expected Outcome:** sale_category is 'Medium Value' for 1000, 'Low Value' for 500.

## Test Case 6
**Test Case ID:** TC06  
**Description:** Error handling - Invalid data type in sale_metadata:discount_applied (e.g., string instead of boolean).  
**Expected Outcome:** Sale is excluded due to failed boolean cast.

## Test Case 7
**Test Case ID:** TC07  
**Description:** Edge case - Empty dataset (no customers, sales, or products).  
**Expected Outcome:** Query returns empty result set.

## Test Case 8
**Test Case ID:** TC08  
**Description:** Edge case - Multiple customers with same total_sales in a region (test RANK window function).  
**Expected Outcome:** Customers with same total_sales get consecutive ranks; top 5 filter works.

## Test Case 9
**Test Case ID:** TC09  
**Description:** Edge case - Customer with multiple products in product_list aggregation.  
**Expected Outcome:** product_list contains all distinct product_ids.

## Test Case 10
**Test Case ID:** TC10  
**Description:** Error handling - Sale with sale_date outside 2023.  
**Expected Outcome:** Sale is excluded from all aggregations and performance CTE.

top_customers AS (

        customer_name,
        region,
        loyalty_level,
        total_sales,
        total_orders,

    FROM 
        customer_sales
),
sales_performance AS (
    SELECT
        s.sale_id,
        s.sale_date,
        s.sale_amount,
        s.discount_percentage,

        `{dataset}.Sales` s
    INNER JOIN 
        `{dataset}.Products` p ON s.product_id = p.product_id

        s.sale_date >= '2023-01-01'

)
SELECT

    tc.total_sales,

    sp.product_name,
    sp.sale_category,
    sp.source
FROM 
    top_customers tc
LEFT JOIN 
    sales_performance sp ON tc.customer_id = sp.customer_id
WHERE 
    tc.region_rank <= 5
ORDER BY

"""
