- **Table Name**: PRODUCT
  - **Attribute Name**: PRODUCT_ID
  - **Data Quality Rule Type**: UniquenessCheck
  - **Rule Description**: Ensure each product has a unique identifier to avoid duplicate records.
  - **SQL**:
    ```sql
    SELECT PRODUCT_ID
    FROM PRODUCT
    GROUP BY PRODUCT_ID
    HAVING COUNT(PRODUCT_ID) > 1;
    ```

  - **Attribute Name**: PRODUCT_NM
  - **Data Quality Rule Type**: NullCheck
  - **Rule Description**: Ensure product name is not null for proper identification.
  - **SQL**:
    ```sql
    SELECT PRODUCT_ID, PRODUCT_NM
    FROM PRODUCT
    WHERE PRODUCT_NM IS NULL;
    ```

  - **Attribute Name**: PRODUCT_NO
  - **Data Quality Rule Type**: FormatCheck
  - **Rule Description**: Ensure product number contains only numeric characters.
  - **SQL**:
    ```sql
    SELECT PRODUCT_ID, PRODUCT_NO
    FROM PRODUCT
    WHERE REGEXP_LIKE(PRODUCT_NO, '[^0-9]');
    ```

  - **Attribute Name**: COLOR
  - **Data Quality Rule Type**: ValueSetCheck
  - **Rule Description**: Ensure color value is from a predefined list of valid colors.
  - **SQL**:
    ```sql
    SELECT PRODUCT_ID, COLOR
    FROM PRODUCT
    WHERE COLOR NOT IN ('Red', 'Blue', 'Green', 'Black', 'White', 'Yellow', 'Orange', 'Purple');
    ```

  - **Attribute Name**: STD_COST
  - **Data Quality Rule Type**: RangeCheck
  - **Rule Description**: Ensure standard cost is non-negative.
  - **SQL**:
    ```sql
    SELECT PRODUCT_ID, STD_COST
    FROM PRODUCT
    WHERE STD_COST < 0 OR STD_COST IS NULL;
    ```

  - **Attribute Name**: LIST_PRICE
  - **Data Quality Rule Type**: RangeCheck
  - **Rule Description**: Ensure list price is greater than or equal to standard cost and non-negative.
  - **SQL**:
    ```sql
    SELECT PRODUCT_ID, LIST_PRICE, STD_COST
    FROM PRODUCT
    WHERE LIST_PRICE < 0 OR LIST_PRICE < STD_COST OR LIST_PRICE IS NULL;
    ```

  - **Attribute Name**: SELL_ST_DT
  - **Data Quality Rule Type**: DateCheck
  - **Rule Description**: Ensure the sell start date is not in the future.
  - **SQL**:
    ```sql
    SELECT PRODUCT_ID, SELL_ST_DT
    FROM PRODUCT
    WHERE SELL_ST_DT > CURRENT_DATE;
    ```

  - **Attribute Name**: SELL_ED_DT
  - **Data Quality Rule Type**: DateCheck
  - **Rule Description**: Ensure the sell end date is after the sell start date.
  - **SQL**:
    ```sql
    SELECT PRODUCT_ID, SELL_ST_DT, SELL_ED_DT
    FROM PRODUCT
    WHERE SELL_ED_DT < SELL_ST_DT;
    ```
