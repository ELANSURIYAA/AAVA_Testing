2. **Pytest Script for each test case:**  
   Below is an example Pytest script for one of the test cases (TC002 - Join and merge logic validation). Similar scripts can be written for the other test cases.

```python
import pytest
import pandas as pd

@pytest.fixture
def mock_input_data():
    # Mock input datasets
    df1 = pd.DataFrame({
        'id': [1, 2, 3],
        'value': ['A', 'B', 'C']
    })
    df2 = pd.DataFrame({
        'id': [1, 2, 4],
        'description': ['Alpha', 'Beta', 'Delta']
    })
    return {"input1": df1, "input2": df2}

def test_join_logic(mock_input_data):
    # Perform the join operation
    result = pd.merge(mock_input_data["input1"], mock_input_data["input2"], on="id", how="inner")
    
    # Expected output
    expected_result = pd.DataFrame({
        'id': [1, 2],
        'value': ['A', 'B'],
        'description': ['Alpha', 'Beta']
    })
    
    # Validate the result
    pd.testing.assert_frame_equal(result, expected_result)
```
